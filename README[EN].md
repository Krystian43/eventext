# eventext
Displays rotating text that can be customized. <br>
features:
* creating own text (one line)
* adjusting text material
  * steel - reflects light like mirror
  * gold - second preset for metal with different color
  * custom steel - reflective surface that can have customized color
  * wood - non reflective material with texture, this also can have custom color
  * plastic - just basic material that can have any color
  * neon - emissive text with any color
* color picker
* scene brightnes
* rotation speed
* size and thicknes
