# eventext
Wyświetla obracający się tekst który można dowolnie ustawiać <br>
Projekt zawiera:
* tworzenie własnego tekstu (1 linia)
* dostosowywanie materiału napisu
  * chrom - połyskujący metaliczny materiał
  * złoto - kolorystyczny preset chromu
  * niestandardowy metal - błyszcząca powieżchnia z odblokowaną zmianą kolorów
  * drewno - matowa powieżchnia z proceduralnie generowaną teksturą słojów, może mieć dowolny kolor
  * plastik - materiał jednolity z drobnymi połyskami, dowolny kolor odblokowany
  * neon - tekst emitujący światło o wybranym koloże
* ustawienie jasności na scenie
* dostosowywanie prędkości obrotu
* dostosowywanie koloru
* dostosowywanie wielkości i grubości
<br>
Projekt został napisany przy użyciu sztucznej inteligencji. Jeśli gdzieś widzisz swój kod napisz w sekcji issues a dodam cię jako współtwórcę lub wzmienie twoją nazwę tutaj.
